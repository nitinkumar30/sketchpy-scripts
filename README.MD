# Sketchpy module

> Using this module, you can get options to get the sketching of some famous characters and actors.  
> Just need to import & start using them.  


# How to install module

```pip install sketchpy```  

OR   

In PyCharm Community  

    CTRL + ALT + S  >>  Open 'Interpreter'   >> Click on +  >> Search for 'sketchpy' & click Install   


# Tools & modules used

> For running the script, you just a modules ```sketchpy```
> For running the script, you need an IDE/IDLE, I'll recommend PyCharm Community.  


# Who're included

1. Robert Downey Junior aka Ironman

3. BTS Actor V(Kim Taehyung)
4. Indian Flag hoisting
